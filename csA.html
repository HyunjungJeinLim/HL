HTML Cheat Sheet

Element Description Example
Text Element
<br/> single line break This is a paragraph.<br/>
<span> inline section of text <span>This is a paragraph.</span>
<p> paragraph <p>This is a paragraph.</p>
Link Element
<a> hyperlink <a href="https://example.com">Visit example.com</a>
List Element
<ul> unordered list <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
<ol> ordered list <ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
<li> list item <li>Item</li>
Select Element
<select> dropdow <select name="cars">
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
</select>
<option> option in a dropdown list <option value="volvo">Volvo</option>
Layout Element
<div> division or section in a document <div>Content here</div>
<span> section in a document <span>Content here</span>
<header> Header of a document or section <header>Header content</header>
<nav> Set of navigation links <nav>Navigation links</nav>
<main> Main content of a document <main>Main content</main>
<section> Section in a document <section>Section content</section>
<article> Independent, self-contained content <article>Article content</article>
<aside> Content aside from the content it is placed in <aside>Aside content</aside>
<footer> Footer for a document or section <footer>Footer content</footer>
<table> HTML table <table>
  <tr>
    <td>Cell 1</td>
    <td>Cell 2</td>
  </tr>
</table>
Form Element
<form> HTML form for user input <form action="/submit" method="post">...</form>
<input> Input control <input type="text" name="fname">
<textarea> multiline input control (text area) <textarea name="message"></textarea>
<button> Clickable button <button>Click me</button>
<label> Label for an <input> element <label for="fname">First Name:</label>
<fieldset> Groups related elements in a form <fieldset>
  <legend>Personal Information</legend>
  ...
</fieldset>
<legend> Caption for a <fieldset> element <legend>Personal Information</legend>

CSS Cheat Sheet

Property Example
color color: blue;
font-size font-size: 16px;
font-family font-family: Arial, sans-serif;
background-color background-color: #f2f2f2;
padding padding: 10px;
margin margin: 20px;
border border: 1px solid #ddd;
text-align text-align: center;
line-height line-height: 2;
width width 200px;
height height: 100px;
Selector Example
ID #SpecialParagraph {
color: red;
}
Class .allIntroductionParagraphs {(or_ p.pink {)
color: pink;
}
Grouping #paragraph1 {
font-size: 40px;
font-weight: bold;
color: green;
}
Universal * {
margin: 0;
padding: 0;
}
Pseudo a:link { background-color: gold; color: green; }
Attribute /* tag with a title attribute */
a[title] {
color: blue;
}

/* tag with an href matching "https://google.com" */
a[href="https://google.com"] {
color: magenta;
}

/* tag with an href containing "google" */
a[href*="google"] {
font-style: italic;
}
Descendant ul.fruits li a {
text-decoration: none;
}
Adjacent h1 + p {
color: green;
font-size: 20px;
text-decoration: underline;
}
Child ul > li {
list-style: disc;
}
:nth-child() /* Selects the second <li> element in a list */
li:nth-child(2) {
color: blue;
}

/* Selects every fourth element among any group of siblings */
:nth-child(4n) {
text-decoration: underline;
}

/* Selects every even element among any group of siblings */
ul:nth-child(even) {
color: gray;
}
